# Disable Automatic Emails

This Odoo 18 Community module disables automatically triggered outgoing emails.

## What it blocks

- Email sends generated from `mail.template`.
- Email notifications generated by `mail.thread` followers and message notifications.
- Queued outgoing mail processed by Odoo's mail queue.
- The standard Odoo mail scheduler is deactivated during module installation.

Blocked queued emails are moved to the `cancel` state instead of being sent.

## Emergency bypass

The module is enabled by the `disable_automatic_emails.enabled` system parameter, which defaults to `True`.
Set it to `False` to restore normal email delivery.

Custom code can opt in to email delivery by using this context flag:

```python
record.with_context(disable_automatic_emails_bypass=True).message_post(...)
```

or:

```python
template.with_context(disable_automatic_emails_bypass=True).send_mail(record.id, force_send=True)
```
