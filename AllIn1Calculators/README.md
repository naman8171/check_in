# All In 1 Calculators

A responsive React + Vite dashboard for commonly used calculators.

## Included calculators

- EMI calculator
- SIP calculator
- GST add/remove calculator
- Discount calculator
- BMI calculator
- Calorie calculator
- Age calculator
- Percentage calculator
- Unit converter

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
