# Modern Report Designer

A Vite + React drag-and-drop report builder. It lets users add report components from a palette, reorder blocks on the canvas, customize selected block content/theme/width, duplicate or delete blocks, and print/export the final report.

Included elements: heading, subheading, text, quote/callout, divider, spacer, metric, progress bar, bar chart, donut chart, table, image/logo placeholder, checklist, timeline, risk badge, and signature.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
