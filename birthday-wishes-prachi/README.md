# Birthday Wishes for Prachi

A beautiful React + Vite birthday landing page made for **Prachi**. It includes a hero wish, celebration cards, responsive design, and code-based image sections where you can update image paths directly in `src/main.jsx`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Add your images

1. Put your photos inside `public/images/`.
2. Open `src/main.jsx`.
3. Change the `src` values in `heroPhotos`, `memoryPhotos`, and `scatteredPhotos`.

Example:

```js
{ src: '/images/my-prachi-photo.jpg', title: 'Best Smile', note: 'My custom note' }
```

## Features

- Premium animated birthday landing page design for Prachi
- Code-based photo paths for hero collage, floating photos, and memory gallery
- Hinglish birthday wishes, surprise card, marquee wishes, and celebration sections
- Confetti rain, floating balloons, glowing name text, animated cake candles, and hover effects
- Fully responsive layout for mobile, tablet, and desktop
- Built with React and Vite
