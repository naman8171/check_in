# Birthday Wishes for Prachi

A beautiful React + Vite birthday landing page made for **Prachi**. It includes a hero wish, celebration cards, responsive design, and an image upload gallery so you can add personal birthday photos directly from the browser.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Premium birthday landing page design for Prachi
- Hinglish birthday wishes and celebration sections
- Upload multiple images and preview them instantly
- Fully responsive layout for mobile, tablet, and desktop
- Built with React and Vite
